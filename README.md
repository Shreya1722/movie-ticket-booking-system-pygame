# Movie Ticket Booking System using Pygame

## Overview
This project is a desktop-based movie ticket booking system developed using Python and Pygame. It allows users to browse movies, select theaters, book tickets, choose food items, and manage booking details.

## Features
- Movie selection interface
- Theater and timing selection
- Ticket booking functionality
- Food ordering system
- MySQL database integration
- User interaction through Pygame GUI

## Technologies Used
- Python
- Pygame
- MySQL

## Key Concepts Implemented
- Event-driven programming
- GUI development
- Database operations
- File handling
- User input validation

## Project Status
This project was developed as an academic learning project and may require additional optimization and debugging for production-level usage.
